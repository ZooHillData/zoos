# Zoos

Zoo Hill Data monorepo.

- Provides low level tools and components to develop web applications with React.
- Builds on top of [Tanstack](https://tanstack.com), [Shadcn](https://ui.shadcn.com) and more

## Getting Started

1. `npm install`
2. `npx nx serve zoos`
